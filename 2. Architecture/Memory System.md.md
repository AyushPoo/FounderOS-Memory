# Memory System

## Problem
LLMs are stateless → lose context across sessions

## Solution
Use Obsidian as persistent memory layer

## How it Works (Planned)
- Inputs captured (Telegram, Browser, LLM chats)
- Sent to n8n
- Processed (summarized, structured)
- Stored in Obsidian

## Types of Memory
- Ideas
- Conversations
- Logs
- Learnings

## Future
- Retrieval system for LLM context injection
