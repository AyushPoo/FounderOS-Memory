# Target Architecture

## Goal
Create a fully autonomous system with memory and execution capability

## Desired Flow
User (Telegram)
    ↓
n8n (orchestrator)
    ↓
Agents (LLMs + tools)
    ↓
Memory Layer (Obsidian)
    ↓
Execution (code / workflows)

## Key Components
- Input Layer (Telegram / Browser)
- Processing Layer (n8n + LLMs)
- Memory Layer (Obsidian)
- Execution Layer (agents)

## Missing Pieces
- Automatic memory ingestion
- Context retrieval system
- Agent coordination
