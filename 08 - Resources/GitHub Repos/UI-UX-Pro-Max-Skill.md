# UI-UX Pro Max Skill

**Repo:** https://github.com/nextlevelbuilder/ui-ux-pro-max-skill  
**Author:** nextlevelbuilder  
**Type:** AI Skill — Design Intelligence System  
**Relevance:** Website, product pages, landing pages, dashboards

## What It Does
Gives any AI coding agent professional UI/UX design intelligence. Automatically generates a complete design system (colors, typography, layout pattern, effects, anti-patterns) based on your product type — before writing a single line of code.

## The Design System Generator
Analyses your request → matches against 161 product categories → outputs:
- **Pattern** — landing page structure (Hero-Centric, Conversion-Optimized, etc.)
- **Style** — UI style (Glassmorphism, Minimalism, Bento Grid, etc.)
- **Color Palette** — industry-specific (161 palettes)
- **Typography** — 57 curated font pairings
- **Anti-patterns** — what NOT to do for your industry
- **Pre-delivery checklist** — accessibility, responsive, hover states, etc.

## Database Scale
- 67 UI styles
- 161 color palettes  
- 57 font pairings
- 99 UX guidelines
- 161 reasoning rules (industry-specific)
- 13 tech stacks supported (React, Next.js, Tailwind, Vue, SwiftUI, Flutter, etc.)

## Finance/SaaS Specific (Relevant for Founder Systems)
Built-in rules for:
- Fintech/Crypto
- Banking / Insurance
- Personal Finance Tracker
- Invoice & Billing Tool
- B2B SaaS
→ Anti-patterns for finance: avoid bright neon, harsh animations, AI purple/pink gradients

## When to Use (For Founder Systems)
- **ALWAYS use this before building any page** on foundersystems.in
- Use it when the Product Builder generates a web_app — inject design system before coding starts
- Use it for the upcoming products (Investor CRM, Founder Dashboard, etc.)
- Run the Python search script to get design system: `python3 search.py "fintech SaaS dashboard" --design-system`
