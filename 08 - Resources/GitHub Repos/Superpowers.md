# Superpowers

**Repo:** https://github.com/obra/superpowers  
**Author:** Jesse Vincent (Prime Radiant)  
**Type:** Agentic Skills Framework / Software Development Methodology  
**Relevance:** Planning, Building, Coding Agents

## What It Does
A complete software development workflow for coding agents (Claude Code, Cursor, Codex). Makes agents step back and *plan before coding* — forces spec → design review → implementation plan → subagent execution → TDD review.

## Key Skills Included
- **brainstorming** — Refines ideas through questions before writing code. Saves design doc.
- **writing-plans** — Breaks work into 2-5 min tasks with exact file paths + verification steps
- **subagent-driven-development** — Dispatches fresh subagent per task with 2-stage review
- **test-driven-development** — Enforces RED→GREEN→REFACTOR cycle strictly
- **systematic-debugging** — 4-phase root cause process
- **requesting-code-review** — Reviews against plan, blocks progress on critical issues
- **finishing-a-development-branch** — Merge/PR/discard decision workflow

## Core Philosophy
- Spec before code — always
- TDD enforced, not optional
- YAGNI + DRY
- Evidence over claims (verify before declaring done)
- Complexity reduction as primary goal

## When to Use (For Founder Systems)
- Any time we plan a new feature or product build
- Use *brainstorming* skill pattern when taking an idea from Ayush
- Use *writing-plans* when breaking down build tasks for the Product Builder
- Reference *systematic-debugging* when Builder - Web App or other workflows break
